Snap-It - Final Year Project

A narrative photography game built in C++ and SFML, where photographing NPCs drives through four narrative phases. NPC movement is fully decoupled from the NPC class via a swappable Strategy-pattern behaviour system (Wander, Flee, Flock, and two state-machine behaviours), while a Factory Method spawns phase-specific "special entities" that a dedicated PhaseManager tracks independently of the game loop.

Are You Afraid Of The Dark - Mobile Game

A top-down survival horror game for iOS and Android, built in Godot with a five-person team over a 12-week agile production cycle with weekly milestones. Players navigate dark levels with a depleting torch while an enemy AI hunts them whenever they're not lit up, with a full post-level analytics pipeline tracking deaths, session time, and player pathing to CSV.

Mirror Multiplayer Tag Game - Unity Networking

A server-authoritative multiplayer tag game built in Unity with Mirror, featuring Runners vs Taggers across rotating maps. Implements Network Transform interpolation for smooth movement, threshold based position syncing to reduce bandwidth, and server side hit validation to prevent fake hit and teleport exploits. Tested under simulated high-latency conditions using Clumsy.

Maze Pathfinding (BFS, A*) - C++ SFML

A templated graph structure (weighted nodes and arcs via adjacency lists) built to implement and visually compare two pathfinding algorithms, Breadth-First Search and A*, with nodes coloured live during the search to show expansion, cost accumulation and the final shortest path.
